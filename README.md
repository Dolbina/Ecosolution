# Ecosolution
### About the project
This application was created for a company Ecosolution, that develops and implements renewable, environmentally friendly energy sources.

Section About presents the main values of the company.

In section Cases there is a slider that presents successful cases of the company.

Frequently Asked Questions are presented in section FAQ.

### Functional capabilities:

* Each item of the burger menu is clickable, and when you click on it, a smooth scroll will take place to the corresponding section on the page.

* When you click on Get in touch, there is a smooth scroll to the Contact Us section.

* Section Cases presents the Infinite Slider. You can change the slide by clicking on the appropriate buttons or by dragging.

* In section FAQ, clicking on the question opens the answer.
